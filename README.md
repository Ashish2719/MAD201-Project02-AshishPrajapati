## Project Description
This is a budget tracking mobile application built with React Native and Expo. [cite_start]It allows users to record income and expenses (CRUD operations).

The app demonstrates several core concepts:
* **Navigation:** Uses combined Stack and Tab navigation[cite: 7].
* **State:** Managed using `useState`[cite: 8].
* **Storage:** Data is saved locally using **AsyncStorage**[cite: 9].
* **API:** Fetches live currency rates for conversions[cite: 10].

## How to Run
1.  Open the terminal in the project folder.
2.  Run `npm install` (to install dependencies).
3.  Run `npm start` to launch the app.
